# Coding Standards

This generic template is intentionally minimal.

`pp-init` should generate `plan/AGENTS.md` from a language profile in:
- `skills/pp-init/assets/languages/python/agents-template.md`
- `skills/pp-init/assets/languages/arduino/agents-template.md`

If this file is used as a fallback, immediately replace it with the selected
language-specific template.
